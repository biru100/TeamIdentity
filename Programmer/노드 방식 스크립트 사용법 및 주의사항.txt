nodescript 에셋을 클릭하면 에디터창 열림

에디터창 영역에 우클릭시 만들수 있는 함수 노드 표시 후 
원하는 함수 선택하여 생성

노드 제작 완료후 save 버튼 누르면 에셋에 저장

노드 창에 Start, Update, Finish 노드는 1개 이상 두지 않는다.

노드이름 바로옆에 있는 포트들은 플로우 관련 노드이다.
(Start, Update, Finish 부터 1개의 플로우씩 코드 진행 됨)

Compile Code 버튼을 눌러서 컴파일 할 것
